# BMT Frontend Domains & Features

This directory organizes application features by business domain boundaries:

*   `authentication/` — Login, registration, email confirmation, and session recovery views.
*   `workspace/` — Tenant selectors and workspace settings.
*   `marketplace/` — Template listings, version rollbacks, and installs.
*   `recommendations/` — AI advisor optimization scorecards.
*   `notifications/` — Timezone-compliant email/Slack settings.
*   `automation/` — Topological rules designers.
