# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not open a public issue. Email your findings to security@bmt.io instead.

We aim to acknowledge all reports within 48 hours and release patch fixes within 7 days.
