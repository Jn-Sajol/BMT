# BMT — Release & Verification Roadmap

This roadmap lists milestones, target release metrics, and verification steps.

## 1. Milestones Schedule

### Milestone `v1.0.0-rc-1` (Code Freeze)
*   **Focus:** Verify DI graphs, lock the API contract, and complete final security audits.
*   **Verification:** Turbo compilation runs, Zod startup checks, and Prometheus metrics verify cleanly.

### Milestone `v1.0.0` (Production Release)
*   **Focus:** Deploy multi-stage Docker configurations and Helm manifests to Kubernetes clusters.
*   **Verification:** Rollback procedures and disaster recovery checklists verify cleanly.

### Milestone `v1.1.0` (Feature Extensions)
*   **Focus:** Support custom AI recommendation models and Google Ads adapters.
