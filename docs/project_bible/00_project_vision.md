# BMT — Project Vision & Core Mission

## 1. Product Statement
The **Business Marketing Tool (BMT)** is an enterprise-grade multi-tenant marketing operating system designed to orchestrate campaigns, run rule-based automation workflows, and serve optimization recommendations across major ad networks (Meta Ads, Google Ads).

---

## 2. Core Segregations: SAFE vs. ADVANCED

```
                      [ BMT Platform Core ]
                                |
               +----------------+----------------+
               |                                 |
         [ SAFE Mode ]                   [ ADVANCED Mode ]
               |                                 |
        (Official APIs)                    (Automations)
        (Provider-Compliant)               (Custom Scripts)
        (Read-Only Insights)               (Marketplace Hub)
```

### SAFE Workspace
Contains only official, provider-compliant features. Focuses on read-only campaign management, metrics aggregation, analytics visualization, and performance dashboards.

### ADVANCED Workspace
Contains automation and optimization logic. Focuses on rule engines, topological workflow execution compilers, custom script installations, and automated budget adjustment actions.
